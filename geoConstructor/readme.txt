Проект geoConstructor представляет собой скрипт на языке Python 3.6, позволяющий автоматизировать процесс создания объемной трёхмерной геометрической модели грунта в графическом редактора GeoPhysicsCAD. В проекте geoConstructor используется для достижения поставленной компьютерное зрение, библиотека openCV для работы с изображениями, библиотека numpy для работы с математическими алгоритмами.
Также в работе используются изображения сечений образца, полученные путем сейсморазведки.
План выполнения задачи:
1. Установка библиотек, загрузка изображения astr_1_1.jpg 883*662 пикселей и изменение оттенков изображения на оттенки серого.
2. Обработка изображения через градиент контрастности для получения характерных линий разделения слоев грунта.
3. Определение местоположения узлов на линии раздела из расчета 20 узлов с равномерным шагом по всей линии раздела (узлы ставятся на сечениях модели, поэтому на 20 сечениях 20 узлов).
4. Выгрузить сетку из узлов с центрально равномерным шагом в файл .txt.
5. Загрузить файл с узлами в программу GeoPhysicsCAD и выполнить дальнейшее построение - соединить узлы линейными сплайнами.
6. В результате должна получиться модель из треугольных и четырёхугольных конечных элементов (зависит от того, как будут соединены узлы).
